Separação entre estrutura e Estilo

Introdução

Tarefa

Seu desafio é construir uma tabela de inventário de produtos. Ela deve ser visualmente organizada e tecnicamente acessível, utilizando cabeçalhos e legendas apropriadas [15, 16].

Objetivo: Replicar uma tabela com 4 colunas (Produto, Categoria, Estoque, Preço) e 3 linhas de dados, incluindo uma linha de rodapé com o total.

Processo

  1. Inicie a estrutura com a tag <table> [11].
  2. Adicione uma legenda descritiva usando <caption> [15].
  3. Crie o cabeçalho com <thead> e utilize <th> com o atributo scope="col" para acessibilidade [18, 19].
  4. Preencha o corpo (<tbody>) com os dados dos produtos usando <td> [6, 12].
  5. Utilize <tfoot> para exibir o somatório do estoque [20].

Recursos

  • MDN: O básico de tabelas HTML [22]
  • WPT: Guia de Tabelas Acessíveis [3]
  • Homehost: Tutorial completo de tags estruturais [23]

Avaliação

Critério Iniciante Excelente
Estrutura HTML Tags básicas sem semântica. Uso correto de table, tr, th, td [6].
Acessibilidade Sem legenda ou scope. Inclui caption e scope nos cabeçalhos [15, 18].
Semântica Não usa thead/tbody. Uso completo de tags estruturais [19, 20].
